# Standard Operating Procedures — inteteam-garage

## New Feature

1. Create `docs/features/{name}/README.md` with user stories + acceptance criteria
2. Create `docs/features/{name}/architecture.md` with technical decisions
3. Create `docs/features/{name}/COMPONENT_INVENTORY.md` — list existing components before planning new ones
4. Create `docs/database/migrations/NNN_{table}.md` for each new table
5. Run `integen laravel:resource {Name}` for standard CRUD resources (see `docs/tasks.md` for field specs)
6. Apply scope rename: `grep -rl "company_id\|HasCompanyScope" app/ database/ | xargs sed -i 's/company_id/garage_id/g; s/HasCompanyScope/HasGarageScope/g'`
7. Write failing tests first
8. Implement backend (migration → model → service → controller)
9. Implement frontend (pages, reuse existing components)
10. Run quality gates (see CLAUDE.md)

## Modifying Existing Code

1. Read the file fully first
2. Check props/types have the data you need
3. Match existing patterns in the file
4. Run quality gates after

## State Machine Changes

All state transitions go through `JobStateMachine::transition()`. Never update `Job.state` directly.
Each transition must:
- Be in the allowed transitions map
- Pass its guard (if any)
- Log to `JobStateTransition`

## Audit Log

All writes to `ApprovalEvent` must go through `ApprovalEventService::record()`.
Never instantiate `ApprovalEvent` directly. Never update or delete events.

## Portal Token Endpoints

Portal routes use `portal.token` middleware. Token is resolved to a `Job` via `SignedPortalTokenService`.
All portal write endpoints (approve, decline, handover, preference) append to `ApprovalEvent` log.

## GCS Uploads

All GCS operations go through `GcsService`. Never call GCS SDK directly in controllers.
Object names must follow the format: `{job_id}/{stage_name}/{timestamp}_{filename}`.
Stage lock is enforced before any upload attempt.

## Debugging

```bash
# Tail logs
docker compose logs -f php-fpm

# Tinker
docker compose exec php-fpm php artisan tinker

# Test a specific class
docker compose exec php-fpm php artisan test --filter=JobStateMachineTest
```
